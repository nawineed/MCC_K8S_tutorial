# MCC_K8S_tutorial

--------------------------------------------------------------------------------------
System Requirements :::
--------------------------------------------------------------------------------------
* CPU: 2 Cores (vCPUs). Standard Kubernetes will typically fail to initialize with only 1 core.
* RAM: 2 GB. However, 4 GB is strongly recommended to handle both system overhead and small application workloads.
* Storage: 20 GB of free disk space. This is required for container images, system logs, and Kubernetes binaries.
* Swap: Disabled. Kubernetes requires swap to be completely turned off to ensure predictable resource scheduling.
  
* For this LAB - Install Ubuntu 22.04.5 LTS machines (2 cores vCPUs, 8GB RAM, 50GB HDD) ----*
